# UI
- リッチUIからワンタップで取得する
- 定時で情報を流す
- 管理画面でいろいろ設定したい

## 管理画面
- 現在配信中の情報を一覧で見れるようにする
- 取得したい値を操作できる
- 配信時間をコントロールできる
